# Documentation Assets

This folder contains documentation assets for the project.

## Screenshots

Add your screenshots here:
- `screenshot-main.png` - Main dashboard view
- `screenshot-charts.png` - Chart visualizations
- `screenshot-animation.png` - Bar chart animation
- `screenshot-mobile.png` - Mobile responsive view

## Creating Screenshots

### Recommended Tools
- **Windows**: Snipping Tool, ShareX
- **macOS**: Screenshot (Cmd+Shift+4)
- **Linux**: GNOME Screenshot, Flameshot

### Guidelines
- Resolution: 1920x1080 or higher
- Format: PNG for UI, JPG for photos
- Optimize with [TinyPNG](https://tinypng.com/)
- Show real data and interactions

## Creating GIFs

### Tools
- [ScreenToGif](https://www.screentogif.com/) (Windows)
- [LICEcap](https://www.cockos.com/licecap/) (Cross-platform)
- [Kap](https://getkap.co/) (macOS)

### Guidelines
- Duration: 5-10 seconds
- FPS: 15-20
- Size: < 5MB
- Show key features in action

## Logo

If you create a logo, add it here as `logo.png` or `logo.svg`

Recommended size: 200x200px or 512x512px
